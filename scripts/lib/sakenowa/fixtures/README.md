# さけのわ API フィクスチャ

テスト用の保存済み実レスポンス（TEST_PHILOSOPHY のフィクスチャ方針）。

- 取得日: 2026-07-04（実 API `https://muro.sakenowa.com/sakenowa-data/api/` から取得）
- 全件は大きいため代表サンプルに間引いている（areas のみ全 48 件）。
  サンプルには次の例外ケースを意図的に含める:
  - `areaId=0`（その他）の蔵元とその銘柄（インポートのスキップ対象）
  - 空文字名の蔵元（プレースホルダとみられる。スキップ対象）
  - フレーバーチャート欠損の銘柄／`tagIds` が空配列の銘柄
- 間引き方法: ランキング上位銘柄＋チャート欠損銘柄＋例外ケースを選び、
  それらに関係する蔵元・チャート・タグ・ランキング行だけを残す。
- 再生成する場合は同じ方針でサンプリングし、`schemas.test.ts` が通ることを確認する。

データ出典: [さけのわデータ](https://sakenowa.com)
